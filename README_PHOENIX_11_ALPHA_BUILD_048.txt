PHOENIX 11 ALPHA · BUILD 048
FOCUS-022 · MODOS AUTOMÁTICO / LIBRE

Implementado exclusivamente en esta build:
- El acceso principal de Home se denomina FOCUS.
- Antes de entrar en el ejercicio se elige AUTOMÁTICO o LIBRE.
- Automático conserva el orden de la rutina y el resumen entre ejercicios.
- Libre permite elegir cualquier ejercicio pendiente.
- Al completar un ejercicio en Libre se vuelve directamente a pendientes.
- En Libre no aparece el resumen intermedio de ejercicio.
- Al terminar todos los ejercicios, ambos modos usan el mismo reporte final.
- El modo elegido se guarda en la sesión como executionMode.
- Sesiones activas antiguas se interpretan como Automático para mantener compatibilidad.
- Se conserva DATA-014: identidad real del ejercicio en cada serie.

No incluido todavía:
- Temporizador propio para ejercicios medidos por tiempo.
- Rediseño Phoenix Light / limpieza visual general.
