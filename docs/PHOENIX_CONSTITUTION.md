# CONSTITUCIÓN DE PHOENIX

1. **Personal First.** Phoenix se construye primero para el uso real de Alberto.
2. **Entrenar primero.** Nada debe distraer del entrenamiento.
3. **Nunca perder una serie.** Cada serie se guarda inmediatamente.
4. **Offline primero.** El entrenamiento no depende de Internet.
5. **Los datos son del usuario.** Siempre exportables, recuperables y portables.
6. **Menos toques.** Cada interacción debe justificar su existencia.
7. **No escribir en el gimnasio.** Teclado solo cuando sea imprescindible.
8. **Una mano.** Los controles principales deben ser cómodos con el pulgar.
9. **Un botón rey.** Cada pantalla tiene una acción principal clara.
10. **Belleza funcional.** La estética FORGED ayuda a usar la app; no es adorno.
11. **Diseño aprobado, diseño bloqueado.** No se cambia una pantalla aprobada sin autorización explícita.
12. **Evolución sin romper.** Cada nueva versión mantiene datos, flujo y funciones anteriores.
13. **Sin publicidad ni suscripciones obligatorias.**
14. **Personal First, Commercial Ready.**
