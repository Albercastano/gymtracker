# Cómo pasar ejercicios manuales

Rellena `templates/PEDB_Ejercicios_Manuales.csv`. Una fila por ejercicio.

Obligatorios: `name_es`, `primary_muscle`, `target_zone`.

Recomendados: `equipment`, `movement_pattern`, `exercise_type`, `home_suitable`, `level`.

Opcional: `parent_exercise_id` si conoces el PEX oficial equivalente.

Para varios materiales usa ` | `. Después sube el CSV y escribe: “Integra estos ejercicios personales en PEDB y genera sus relaciones”.
