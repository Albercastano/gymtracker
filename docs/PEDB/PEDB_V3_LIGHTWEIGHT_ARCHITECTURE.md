## ADN compacto
Cada ejercicio contiene:

`dna = [dificultad_técnica, estabilidad, fatiga, máscara_objetivos]`

## Relaciones
Las claves compactas son:
- `s`: origen
- `t`: destino
- `c`: categoría
- `r`: recomendada
- `p`: puntuación
- `f`: confianza

La aplicación muestra las 3 recomendadas y consulta el resto bajo demanda.
