# Integración PEDB v3.1

1. Sustituir el módulo PEDB anterior.
2. Incrementar la versión de IndexedDB.
3. Cargar los nuevos archivos sólo cuando se use su función.
4. Usar `w` en las relaciones para mostrar «¿Por qué este ejercicio?».
5. Usar `packs.json` como filtros virtuales; no copiar ejercicios.
6. Guardar perfiles de equipamiento como datos del usuario, nunca dentro de PEDB.
7. Mantener ejercicios `USR-EX` separados y recalcular sus relaciones.
