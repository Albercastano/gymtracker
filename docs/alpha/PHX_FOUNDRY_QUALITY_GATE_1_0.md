# PHX Foundry Quality Gate 1.0

**Build:** Phoenix 11 Alpha Build 007

## Objetivo

Ningún material se extiende a toda Phoenix hasta superar una puerta de calidad visual, responsive, funcional y de seguridad.

## Niveles visuales

- **Hero:** Timer, Hoy toca, PR y resumen final. Máxima presencia.
- **Structural:** rutinas, PEDB, métricas y planificación. Masa contenida.
- **Quiet:** ajustes secundarios, textos y listas largas. Fondo silencioso.

## Estados obligatorios

Normal, pulsado, seleccionado, foco, cargando, desactivado, éxito y error.

## Seguridad de material

- Sin JavaScript.
- Sin conexiones.
- Sin lectura de datos.
- Precision como fallback permanente.
- Vista previa no persistente antes de aplicar.

## Quality Gate automático

Comprueba contrato, seguridad, áreas táctiles, overflow, componentes semánticos, ausencia de diálogos nativos, persistencia controlada y recorte de texto crítico.
