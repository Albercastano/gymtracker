# PHX FORGE LAB · Build 002

## Objetivo

Forge Lab es el banco interno de pruebas visuales de GymTracker Phoenix. Renderiza los mismos componentes bajo FORGED Precision y FORGED Foundry sin tocar datos de usuario ni lógica de entrenamiento.

## Acceso

`DATOS → AJUSTES → APARIENCIA Y MATERIALES → ABRIR FORGE LAB`

## Incluye

- Selector inmediato Precision / Foundry.
- Botones primario, secundario, destructivo y desactivado.
- Tarjetas, métricas, progreso y estados.
- Inputs, selector segmentado y switch.
- Specimen del PHX Timer.
- Ficha PEDB.
- Estados guardado, advertencia, error y vacío.
- Diagnóstico de viewport, orientación, texto y material.

## Reglas

1. Forge Lab no guarda los valores de muestra.
2. No contiene datos reales del perfil.
3. No realiza conexiones.
4. Los materiales solo cambian presentación.
5. Precision sigue siendo el fallback obligatorio.

## Uso futuro

Cada material creado por Phoenix Forge deberá validarse en Forge Lab antes de poder considerarse compatible con una versión del PHX Skin Engine.
