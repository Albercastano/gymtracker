# PHX SKIN SDK · Inicio rápido

1. Copia `themes/_template`.
2. Cambia `id`, `name`, `author` y `version` en `manifest.json`.
3. Diseña usando tokens semánticos y componentes `phx-*`; no apuntes a pantallas concretas ni escribas lógica de entrenamiento.
4. No uses URLs, `@import`, JavaScript ni recursos remotos.
5. Ejecuta el validador offline.
6. Prueba la creación en Forge Lab en vertical, apaisado y texto grande.
7. Declara `engine: 0.3.x` y `componentContract: 0.1.x`.
8. Solo después se añade al registro de materiales instalados.

## Filosofía

Una Creación de la Forja cambia el material percibido de Phoenix. No cambia datos, navegación, jerarquía funcional ni seguridad.
